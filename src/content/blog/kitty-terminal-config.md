```bash
sudo pacman -S kitty  # Arch installation
```
**CLI Theme Selector**  
   ```bash
   kitty +kitten themes --reload-in=all
   ```
   Choose the **GitHub Light High Contrast** theme and press `M`.

## **Base Configuration (`~/.config/kitty/kitty.conf`)**

### Font Configuration
```conf
font_family      FiraCode Nerd Font
font_size        12.0
```
